# Project Objectives

## 1. CP-ABE Access Control for EMR
- Develop a system that applies Ciphertext-Policy Attribute-Based Encryption (CP-ABE) for providing fine-grained, attribute-based, policy-controlled access to electronic medical records (EMR).

## 2. QR Code-Enabled Mobile EMR Registration
- Utilize a QR Code-enabled mobile EMR registration system to streamline patient onboarding and improve record management accessibility.

## 3. Performance Assessment (OCR)
- Assess the performance of the applied algorithms based on accuracy, F1-score, precision, and recall (OCR).
